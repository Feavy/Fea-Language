# Fea Language (*WIP*)
## A propos
Fea Language est un un langage de programmation user-friendly inspiré de l'ActionScript3 et dont le code est transpilé en C.

Tout est réalisé "from scratch", de la grammaire jusqu'au transpilateur.
### Exemple
```java
main():int[] {
	nb:int = 42;
	if(nb >= 42) {
		varIf:int;
		varIf2:int;
	} else if(nb<30){
		varElse:int;
		varElse2:int;
	}
}
```
